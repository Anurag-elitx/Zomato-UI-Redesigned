## Tech Stack
- React
- Vite
- Vanilla CSS (Variables & Flex/Grid)
- React Icons
- Google Fonts (Outfit & Inter)

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
